CRUD com Java SpringBoot & Angular

Um CRUD (Create, Read, Update, Delete) desenvolvido com Java Spring Boot e Angular envolve a criação de um sistema completo onde o front-end em Angular se comunica com o back-end em Java Spring Boot para gerenciar dados de forma eficiente. O banco de dados utilizado é o JPA com H2, que permite uma integração simples e rápida, sendo ideal para desenvolvimento e testes.
